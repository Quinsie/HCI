# 전자출결 — 아이폰 설치 안내 (사용자용)

이 앱은 테스트용이라 **App Store에 없습니다.** 대신 **설치 담당자의 Mac에 아이폰을 연결**해서 설치합니다.
무료 Apple ID로 서명하므로 **설치 후 7일간** 쓸 수 있고, 만료되면 다시 연결해 재설치하면 됩니다.

> 총 5명(본인 포함). 아래에서 **본인 역할**에 해당하는 부분만 보면 됩니다.

---

## 🅰️ 설치 담당자 (Mac + Apple ID 보유, 1명)

처음 한 번만 환경을 맞춘 뒤, 팀원 아이폰을 차례로 꽂아 설치합니다.

1. **도구 준비(최초 1회)** — Xcode 설치 + 서명 설정. 자세한 절차는 **[iOS_빌드_가이드.md](iOS_빌드_가이드.md)** 참고.
2. **아이폰 한 대 연결** → 그 아이폰에서 "이 컴퓨터를 신뢰" 탭.
3. 터미널에서 한 줄:
   ```bash
   scripts/ios_install.sh                 # 연결된 기기의 device-id 확인
   scripts/ios_install.sh <device-id>     # 그 아이폰에 설치
   ```
4. 케이블을 다음 사람 아이폰으로 옮겨 2~3 반복 (총 5대).
5. **7일 뒤** 누군가 "앱이 안 열려요" 하면, 그 아이폰을 다시 연결하고 3번을 한 번 더.

> 한 Apple ID로 5대 모두 설치하면 서명 설정은 한 번만 하면 됩니다.

---

## 🅱️ 아이폰 사용자 (본인 폰에 받는 5명 각각)

설치 담당자에게 폰을 건네거나 직접 옆에서, 아래만 하면 됩니다. **어렵지 않아요.**

### ① 설치할 때 (담당자 Mac 앞에서)
1. 본인 아이폰을 담당자 Mac에 **케이블로 연결**.
2. 아이폰 화면에 **"이 컴퓨터를 신뢰하시겠습니까?"** → **신뢰** → 폰 암호 입력.
3. 담당자가 설치를 실행하면 잠시 기다립니다(빌드 1~2분).

### ② 설치 직후 — "신뢰" 한 번 (이거 안 하면 앱이 안 열림!)
홈 화면에 **전자출결** 아이콘이 생겼지만, 처음 누르면 *"신뢰되지 않은 개발자"* 경고가 뜹니다.
1. 아이폰 **설정** 앱 열기
2. **일반 ▸ VPN 및 기기 관리**
3. **개발자 앱** 항목에서 담당자의 **Apple ID 이메일** 탭
4. **"...을(를) 신뢰"** → 팝업에서 **신뢰**

### ③ 사용
이제 홈 화면의 **전자출결** 아이콘을 누르면 실행됩니다. 🎉

### ④ 7일 뒤
무료 설치는 **7일 후 만료**되어 앱이 안 열릴 수 있습니다.
그럴 땐 **삭제하지 말고** 설치 담당자에게 말해서 폰을 다시 한 번 연결 → 재설치하면 다시 7일 사용 가능합니다.

---

## 자주 묻는 것
- **데이터 요금/인터넷 필요해요?** 설치는 케이블로 합니다. 앱 자체는 인터넷 없이도 동작(로컬 테스트 빌드).
- **제 폰이 위험해지나요?** 아니요. 본인이 직접 "신뢰"한 개발자(담당자 Apple ID) 앱만 실행됩니다. 7일 뒤 자동으로 못 쓰게 됩니다.
- **안드로이드인데요?** 안드로이드는 APK 파일 하나로 더 간단합니다 — [README](../README.md)의 Android APK 섹션 참고.
